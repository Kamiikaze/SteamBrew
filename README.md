# SteamBrew

SteamBrew is a monorepo for Steam client customizations, including Millennium themes and plugins.


### Themes

| Name        | Description            |
|-------------|------------------------|
| **[Steam-Library Lightbox Images](https://github.com/Kamiikaze/SteamBrew/tree/master/Themes/library-lightbox)** | Adds a fullscreen lightbox image viewer to Steam Library news posts. |


### Plugins

The `Plugins` directory is reserved for Millennium plugin projects.
